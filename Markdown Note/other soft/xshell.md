# xshell使用方法
## 1.ssh隧道
- 如图所示![隧道配置图](/resources/xshell1.gif)
- 此配置是将远程端口8080通过ssh隧道映射至本地端口8080
- 即使远程主机的8080端口没有开放
- 本地也能通过127.0.0.1:8080访问到远程主机的8080端口

## 2.多窗口
